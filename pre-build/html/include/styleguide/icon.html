아이콘 크기 (icon size)

시스템 아이콘은 24px를 기준으로 정의하며, 주로 사용하는 크기는 12px, 14px, 16px, 20px, 24px, 28px, 32px, 40px이다.

아이콘 크기는 icon--12, icon--24, icon--40 등의 클래스로 제어한다.

Html

          <svg class="svg-icon icon--24 color-white" aria-hidden="true">
  <use href="#icon-outline-search"></use>
</svg>
